================================================================================
                            API REFERENCE
================================================================================

BinanceConnector Methods:
  - ping() → Dict (empty dict if successful)
  - get_server_time() → int (milliseconds)
  - get_balance() → Dict[str, float] (asset balances)
  - get_trading_pair_info(symbol) → Dict (market info)
  - place_order(symbol, side, type, qty, price, client_oid) → Dict
  - cancel_order(symbol, client_oid) → Dict
  - get_order_status(symbol, client_oid) → Dict
  - get_open_orders(symbol=None) → List[Dict]
  - get_order_history(symbol=None, limit=100) → List[Dict]

OrderTracker Methods:
  - create_order(...) → InFlightOrder
  - set_exchange_id(client_oid, exchange_oid) → bool
  - add_fill(client_oid, qty, price, fee, fee_asset) → bool
  - cancel_order(client_oid) → bool
  - get_order(client_oid) → InFlightOrder
  - get_order_by_exchange_id(exchange_oid) → InFlightOrder
  - get_open_orders(symbol=None) → List[InFlightOrder]
  - close_order(client_oid) → InFlightOrder
  - get_order_history(symbol=None, limit=100) → List[InFlightOrder]
  - get_stats() → Dict

OrderState Enum (8 states):
  - PENDING: Created locally, not on exchange
  - OPEN: Confirmed by exchange
  - PARTIALLY_FILLED: Partial fill received
  - FILLED: Fully filled
  - CANCELLED: Cancelled by user
  - REJECTED: Rejected by exchange
  - EXPIRED: Expired (GTC timeout)
  - FAILED: Failed to create

================================================================================
                           QUICK START GUIDE
================================================================================

1. Set environment variables:
   export BINANCE_API_KEY=your_testnet_key
   export BINANCE_API_SECRET=your_testnet_secret

2. Import modules:
   from yunmin.connectors.binance_connector import BinanceConnector
   from yunmin.core.order_tracker import OrderTracker

3. Create instances:
   import os
   connector = BinanceConnector(
       api_key=os.getenv("BINANCE_API_KEY"),
       api_secret=os.getenv("BINANCE_API_SECRET"),
       testnet=True
   )
   tracker = OrderTracker()

4. Test connectivity:
   assert connector.ping() == {}
   balance = connector.get_balance()

5. Create and track order:
   order = tracker.create_order(
       client_order_id="test_001",
       symbol="BTCUSDT",
       side="BUY",
       order_type="LIMIT",
       qty=0.1,
       price=42000
   )
   tracker.set_exchange_id("test_001", "exchange_oid")
   tracker.add_fill("test_001", qty=0.1, price=42000, fee=0.05, fee_asset="USDT")

6. Run tests:
   pytest tests/test_binance_connector_integration.py -v -s
